# Creative Media Agency Website

A modern, responsive media agency website showcasing creative work with a streamlined design and dark theme featuring purple/pink/orange accents.

## 🎯 Project Overview

This website is designed for a media agency specializing in:
- **Reels Creation** - Short-form video content for Instagram and TikTok
- **Post Sharing** - Strategic social media content and posting
- **Brand Content** - Comprehensive brand storytelling and visual assets

## ✨ Key Features

### Design & Theme
- **Modern Dark Theme** with sophisticated color palette
- **Purple/Pink/Orange Accents** for visual appeal
- **Responsive Design** optimized for all devices
- **Smooth Animations** and hover effects

### Core Sections
1. **Hero Section** - Animated background with floating gradient orbs
2. **Services** - Three main service packages with pricing
3. **Portfolio** - Featured work showcase with project metrics
4. **Contact** - WhatsApp integration and contact forms
5. **Navigation** - Fixed navbar with smooth scrolling

### Interactive Elements
- **Mobile-responsive navigation** with hamburger menu
- **Portfolio hover effects** revealing project statistics
- **Contact form validation** with notification system
- **Smooth scrolling** between sections
- **Parallax effects** on hero background

## 🚀 Technologies Used

- **HTML5** - Semantic markup structure
- **CSS3** - Modern styling with CSS Grid, Flexbox, and custom properties
- **JavaScript (ES6+)** - Interactive functionality and animations
- **Font Awesome** - Icon library for visual elements
- **Google Fonts** - Inter font family for typography

## 📱 Responsive Features

- **Mobile-first approach** with progressive enhancement
- **Flexible grid layouts** that adapt to screen sizes
- **Touch-friendly interactions** for mobile devices
- **Optimized typography** scaling across devices

## 📋 Customization

### Updating Content
- **Services**: Modify service cards in the HTML
- **Portfolio**: Add/remove portfolio items with appropriate images
- **Contact**: Update WhatsApp number and contact information
- **Branding**: Change colors in CSS custom properties

### Adding Features
- **Portfolio Filtering**: Uncomment filter functionality in JavaScript
- **Contact Form**: Connect to backend service for form submission
- **Analytics**: Add tracking codes for user behavior
- **CMS Integration**: Convert to dynamic content management
